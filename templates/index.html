API Documentation

Overview

This API allows you to get movie recommendations by sending a GET request to the endpoint.

API Endpoint

The API is accessible at: http://localhost:9092/recommend/user_id

Response

The API will return a JSON object containing the recommended movies. 

{
    "recommendations": [
        {
            "movie_id": 1,
            "title": "Movie Title 1",            
            "rating": 4.5
        },
        {
            "movie_id": 2,
            "title": "Movie Title 2",            
            "rating": 4.0
        }
    ]
}
